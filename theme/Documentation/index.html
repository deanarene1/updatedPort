Documentation

Thankyou for purchasing Benzene, HTML Site Template on ThemeForest.

Files Structure

  • Theme Benzene (This folder consists of all required files)
    • /css (Template css)
    • /js (Template js)
    • /php (PHP helpers i.e. Contact Form)
    • /vendor (Consists of all external libs)
    • /images (Consists of Dummy images)
    • /index.html (Template html)
    • /single-portfolio.html (Single Portfolio file)
    • /single-blog.html (Single Blog file)

Note: Images shown in the Template preview are for the preview purpose only and they are not included in the download Theme files.


Installation

  1. Open the Theme Benzene folder to find all the Template Files.
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • Theme Benzene/css
    • Theme Benzene/js
    • Theme Benzene/php
    • Theme Benzene/images
    • Theme Benzene/vendor
    • Theme Benzene/index.html
    • Theme Benzene/single-portfolio.html
    • Theme Benzene/single-blog.html
  4. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

Changing Fonts

Benzene uses Roboto and Montserrat font from the Google Fonts Library. You can find the Linking to the Font Files in the head tag of the index.html file.


Changing font requires following change in all the css files.



Text Animation

Benzene uses Typed jquery plugin for the animated text effects in the home section. To change the texts head over to main.js file and make changes in the following lines.


Carousel for Testimonial section

Benzene uses Owl Carousel for displaying Testimonial section. To use carousel, follow the steps below.

Step 1: Include owl.carousel.min.css, which can be found in the owl-carousel folder in the vendor folder and plugin.js, which can be found in the plugin folder in the vendor folder.

Step 2: Include the html.

Step 3: Include css for Testimonial section available in the external css files.

Step 4: Include script for carousel available in the main.js file.


Carousel for Client section

Benzene again uses Owl Carousel for displaying Client section. To use carousel, follow the steps below.

Step 1: Include owl.carousel.min.css, which can be found in the owl-carousel folder in the vendor folder and plugin.js, which can be found in the plugin folder in the vendor folder.

Step 2: Include the html.

Step 3: Include css for Client section available in the external css files.

Step 4: Include script for carousel available in the main.js file.


Magnific Popup for Portfolio section

Benzene uses Magnific Popup for displaying images in gallery in the portfolio section. To use magnific popup, follow the steps below.

Step 1: Include magnific-popup.css, which can be found in the owl-carousel folder in the vendor folder and plugin.js, which can be found in the plugin folder in the vendor folder.

Step 2: Include the html.

Step 3: Include css for Portfolio section available in the external css files.

Step 4: Include script for magnific popup available in the main.js file.


Benzene uses Mixitup jquery plugin for the filter effects in the portfolio section.

Benzene uses Hoverdir jquery plugin for the hover effects in the portfolio section.


Contact Form

To use contact form, follow the steps below.

Step 1: Include html for the contact from the index.html file.

Step 2: Include css for the contact available in the external css file.

Step 3: Include validate.js script from the validate folder in the vendor folder and the script available in the main.js file for the validation to work.

Step 4: Include php for the contact form available in the contact.php file, just replace personal@website.com with your email address and you are all set.
